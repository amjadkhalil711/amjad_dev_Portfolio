# AMJAD ALSUFYANI — Next‑Gen Terminal Portfolio

A major upgraded portfolio experience:
- Cinematic boot sequence
- Terminal / zsh inspired UI
- Smooth section reveal transitions
- Typing animation per section when the visitor enters it
- Blinking cursor while writing
- 3D tilt glass/terminal cards
- Command Center with Ctrl/⌘ + K
- Skill filters and project filters
- Scroll progress indicator
- Active navigation highlighting
- Accent theme switcher
- Arabic / English language switch
- Animated mesh/grid background
- Responsive mobile layout
- Contact terminal form
- CV download placeholder
